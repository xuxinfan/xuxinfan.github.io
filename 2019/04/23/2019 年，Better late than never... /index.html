2019 年，Better late than never...

四月 23, 2019
  • 生活

虽迟未晚

2019已过半载，入行将近iOS也将近3年了，一直有萌生博客的写法，但都是3分钟热度，简书，掘金都稍有尝试，最开始写的也都是没啥意义的技术文。最近由于公司乔迁，上下班路途遥远，便油然而生了跳槽离职的想法。但在四月份末期的求职之旅中处处碰壁，发现自己很多东西在职业路上捡起来又丢弃了。故有了开始搭建一个专门博客记录我的产出，也让我后头方便回忆的想法，虽然这有点晚了，但我相信 “Better late than never” 。

2019 计划目标

正如上面所述，实际我在编写此篇博文的时候其实2019年已剩下半年，希望从今天开始，打破舒适区，能在剩下的时间完成以下的目标：

优先度高：

  • 使用 Flutter 仿写一个项目，包含正常App所使用的模块与功能。
  • 使用并熟悉Swift，并也有Swift的项目经验（这点可能通过参与公司项目来实现）并学习RxSwft应用到自己可能编写的项目中。
  • 使用 ReactNative 仿写一个项目，包含正常App所使用的模块与功能。
  • RunTime，RunLoop，Block，KVCKVO，源码解析。
  • YYKit 源码解析。

优先度低：

  • 写一个漫画阅读器自用，语言环境待定
  • 成为独立App开发作者，写一个番剧相关的App并发布到AppStore
