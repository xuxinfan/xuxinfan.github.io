安装fastlane及配置环境变量

安装最新的Xcode命令行工具：

xcode-select --install

安装fastlane

# Fastlane其实是一个Ruby脚本的集合
sudo gem install fastlane -NV

# 此处可能需要先安装Homebrew
brew cask install fastlane

设置fastlane

找到项目的目录

fastlane init

此处我们选择手动配置

设置环境变量

FastLane需要设置一些环境变量才能正确运行。特别是，如果没有将区域设置设置为UTF-8区域设置，将导致生成和上传生成的问题。

在shell配置文件中

vim ~/.bash_profile

添加以下行：

export LC_ALL=zh_CN.UTF-8
export LANG=zh_CN.UTF-8
